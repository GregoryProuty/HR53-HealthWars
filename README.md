## Development

### Installing Dependencies

From within the root directory:

```sh
npm install
```

To use dummy data~
Run mongod and insert the dummy data through the pre-built js file.

From root directory:
```
mongod

```
In a new tab:
```
node server/insertDummyData.js
```
